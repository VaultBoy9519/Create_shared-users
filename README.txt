Для работы программы нужно следующее:

1) Ставим node-v18.16.0-x64.msi из текущей папки
2) открываем Пуск - командная строка (Win+R - cmd), пишем команду
npm install cypress

После этих действий можно запускать Create users for shared.bat, вставлять туда ссылку на shared или dev. Создание пользователей займет от 5 до 12 минут в зависимости от скорости работы сервера.

Более подробно можно ознакомиться с инструкциями в видео manual for creator.mp4 в текущей папке.
