Для работы программы нужно следующее:

1) Распаковать архив по пути, не содержащем кириллицу, это важно;
1) Установить node-v18.16.0-x64.msi из текущей папки;
2) Открыть Пуск - командная строка (Win+R - cmd), написать команду
npm install cypress
3) Если после предыдущего пункта ничего не заработало - с помощью команды
cd Путь_к_папке пройти в папку с create_shared_users и установить cypress туда с помощью предыдущей команды.

После этих действий можно запускать Create users for shared.bat, вставлять туда ссылку на shared или dev. 
Выберите пункт, который создаст нужных вам пользователей.

Создание пользователей займет от 5 до 12 минут в зависимости от скорости работы сервера.

Более подробно можно ознакомиться с инструкциями в видео manual for creator.mp4 в текущей папке.
